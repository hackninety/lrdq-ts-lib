# 畢法百法 · 检测口径说明

本库对《畢法賦》百法做机器命中检测，首批实现 **23 法**，其余 77 法以语料形式
收录（`bifa.entries`，含注文与附属格），待后续分批实现。

## 判定级别

- **exact（确判）**：注文条件明确，可字面直判
- **approx（近似）**：判定口径存在取舍（寄宫口径、一词多义等），命中仅供参考

## 输入约定

- 三传六亲由本库依日干五行自算，不依赖宿主命名
- 天将名做归一化（貴人/天一→贵人、腾虵/騰蛇→螣蛇、大裳→太常、勾陣→勾陈、元武→玄武等）
- 未提供完整天地盘（`gong` 十二宫）时，第 51/52 法自动跳过
