# 畢法百法 · 检测口径说明

本库对《畢法賦》百法做机器命中检测，现覆盖 **97/100 法**；不设检测器的 3 法
（第 93/97/98 法）原因见文末。全部百法均以语料形式收录（`bifa.entries`，含注文与附属格）。

## 判定级别

- **exact（确判）**：注文条件明确，可字面直判
- **approx（近似）**：判定口径存在取舍（寄宫口径、一词多义、注文为义理而由格义推定等），命中仅供参考

## 不设检测器的 3 法（UNDETECTABLE_NOS）

| 法号 | 赋句 | 原因 |
|---|---|---|
| 93 | 妄用三傳災福異 | 起传校勘法：判「三传起错」须以本派九宗门重演全式。本库引擎中立，不同流派三传本应分歧（如《占事略決》古法涉害孟仲季与通行不同），机判恐将流派差异误报为「妄用」。宿主的「三传多派对照」功能即此法的现代替代 |
| 97 | 所筮不入仍憑類 | 断法方法论（类神不入课时凭类神言方所色目），无格局条件 |
| 98 | 非占現類勿言之 | 断法方法论（勿言非所占之类），无格局条件 |

## 五行寄旺口径（依各法注文，不强行统一）

- **长生十二宫基表**：水土同长生申（第34法注「申為戊土之長生」），败/旺/死/墓/绝依序推
- **胎财**（第19/20法）：火土同宫（注文「戊己……明知子上是胞胎」），表：戊己子、庚辛卯、壬癸午、甲乙酉
- **绝神**（第79法）：土寄寅故绝亥（注文明言），表：木申、火亥、金寅、水巳、土亥
- **墓**：木未、火戌、金丑、水土辰（通行）

## 输入约定

- 三传六亲由本库依日干五行自算，不依赖宿主命名
- 天将名做归一化（貴人/天一→贵人、腾虵/騰蛇→螣蛇、大裳→太常、勾陣→勾陈、元武→玄武等）
- 未提供完整天地盘（`gong` 十二宫）时，依赖盘位的法（10/38/39/43/45/46/47/49/53/69/73/85/87/92 等）自动跳过
- 昼夜贵人用通行歌诀（甲戊庚牛羊……）；须知昼夜的法（第3法）在 `dayNight` 缺省时跳过
- `nianMing`（本命/行年）为可选输入；第56法等年命类条件缺省时静默跳过
- 第4法与第91法同象异占（鬼乘虎临干），依注文并报，由占事语境取舍
